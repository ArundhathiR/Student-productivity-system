# Student Productivity and Task Recommendation System

## Current Features

- Task class to represent individual tasks
- TaskManager class to manage multiple tasks
- Menu-driven command-line interface
- Add, view, and mark tasks as completed
- Persistent storage using file handling
- Automatic task loading on program startup
- Filter tasks by pending or completed status
- Display productivity insights (total, completed, pending tasks)
- Highlight high-priority pending tasks

## How to Run

1. Clone the repository
2. Navigate to the project directory
3. Run the main Python program:
   ```bash
   python task.py
   ```
4. Choose options from the displayed menu to add, view, or complete tasks
